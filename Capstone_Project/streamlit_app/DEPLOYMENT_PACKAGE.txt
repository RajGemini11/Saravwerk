## Hospital Staffing Prediction - Streamlit Deployment

### 📦 Complete Package Contents

```
hospital-staffing-app/
├── train_and_save_model.py      # Model training and persistence
├── main.py                       # FastAPI backend
├── ui_app.py                     # Streamlit frontend
├── requirements.txt              # Python dependencies
├── hospital_staffing_model.pkl   # Trained model (generated)
├── feature_columns.pkl           # Feature metadata (generated)
├── model_metadata.pkl            # Model info (generated)
└── README_DEPLOYMENT.md          # This file
```

### 🚀 Quick Start (3 Steps)

#### Step 1: Install Dependencies

```bash
pip install -r requirements.txt
```

#### Step 2: Train and Save Model

```bash
python train_and_save_model.py
```

**Output files created:**
- `hospital_staffing_model.pkl` (Ridge Regression model)
- `feature_columns.pkl` (Feature list)
- `model_metadata.pkl` (Model metadata)
- `recent_actual_volumes.csv` (Sample data)

#### Step 3: Run Application

**Terminal 1 - Start FastAPI Backend:**
```bash
uvicorn main:app --reload
```

**Terminal 2 - Start Streamlit UI:**
```bash
streamlit run ui_app.py
```

**Access the app:**
- Streamlit UI: http://localhost:8501
- FastAPI Docs: http://127.0.0.1:8000/docs
- FastAPI Root: http://127.0.0.1:8000

### 🔧 Troubleshooting

**Issue: API not connecting**
```bash
# Check if FastAPI is running
curl http://127.0.0.1:8000/health

# Restart FastAPI
uvicorn main:app --reload
```

**Issue: Model files not found**
```bash
# Retrain model
python train_and_save_model.py
```

**Issue: Port already in use**
```bash
# Change FastAPI port
uvicorn main:app --reload --port 8001

# Update API URL in Streamlit sidebar
http://127.0.0.1:8001
```

### 🌐 Deployment Options

#### Option 1: Local Development (Current Setup)
- FastAPI: localhost:8000
- Streamlit: localhost:8501

#### Option 2: Streamlit Cloud
1. Push to GitHub
2. Connect to Streamlit Cloud
3. Deploy (free tier available)

#### Option 3: Docker Deployment
```dockerfile
FROM python:3.9-slim

WORKDIR /app
COPY requirements.txt .
RUN pip install -r requirements.txt

COPY . .

EXPOSE 8000 8501

CMD ["sh", "-c", "uvicorn main:app --host 0.0.0.0 --port 8000 & streamlit run ui_app.py --server.port 8501"]
```

#### Option 4: Cloud Deployment (AWS/GCP/Azure)
- Deploy FastAPI as containerized service
- Deploy Streamlit as web app
- Use managed database for model storage
